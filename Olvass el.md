Tanulási előrehaladás program használata

A programom a „Tanulási előrehaladás” fantázia nevet viseli, ez egy tanuló program, ahol a felhasználók a programozást próbálhatják ki JavaScript nyelven.
Ebben a dokumentumban elolvashatják, hogy a programot hogyan is kell használni.
Fejlesztőkörnyezet: A kliensoldal HTML, CSS és JavaScript technológiákból áll össze.
A programot Visual Studio Code-ban készítettem.
Visual Studio Code-nak ezt a Windows-os verzióját használtam:
Visual Studio Code 1.111.0: https://code.visualstudio.com/updates/v1_111
(Downloads: Windows: x64)
A Visual Studio Code-ban letöltött bővítmények:
-SQLite Viewer
Elindítás: A program elindításához elegendő a TanulasElorehaladasApp.exe fájlra duplán kattintani.
Az .exe fájlra kattintás hatására megjelenik egy fekete parancssori ablak – ez a szerver, ezt nem kell bezárni. A program automatikusan megnyitja az alapértelmezett böngészőben a http://localhost:3000 címet, ahol a program felülete elérhetővé válik.
Leállítás: A program leállításához a fekete parancssori ablakot kell bezárni. 
Adatbázis: Az adatbázis fájl (app.db) automatikusan létrejön az .exe fájl melletti mappában az első indításkor, és az adatok ott mentődnek.
