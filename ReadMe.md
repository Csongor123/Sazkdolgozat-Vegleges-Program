Using the Learning Progress Program

My program is called “Learning Progress”, it is a learning program where users can try programming in JavaScript.
In this document you can read how to use the program.
Development environment: The client side consists of HTML, CSS and JavaScript technologies.
I created the program in Visual Studio Code.
I used this Windows version of Visual Studio Code:
Visual Studio Code 1.111.0: https://code.visualstudio.com/updates/v1_111
(Downloads: Windows: x64)
Extensions downloaded in Visual Studio Code:
-SQLite Viewer

Start: To start the program, simply double-click on the TanulasElorehaladasApp.exe file.
Clicking on the .exe file will open a black command prompt window – this is the server, you do not need to close it. The program will automatically open the address http://localhost:3000 in the default browser, where the program interface will be available.
Shutdown: To shut down the program, you need to close the black command prompt window.
Database: The database file (app.db) is automatically created in the folder next to the .exe file upon first launch, and the data is saved there.
